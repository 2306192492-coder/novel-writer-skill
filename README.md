# 📖 Novel Writer Skill - 工业级长篇小说创作引擎

> **版本**: v3.1  
> **作者**: Saber (for Hermes Agent)  
> **模型**: 讯飞星辰 `astron-code-latest`  
> **定位**: 支持千万字长篇的自动化小说创作引擎

---

## 🚀 核心功能

### 1. 向量检索引擎 (解决长文记忆)
- **问题**: 写到第 100 章，忘了第 5 章的设定？
- **解决**: 内置轻量级 TF-IDF 语义搜索，写新章节时自动检索相关段落
- **效果**: 只加载最相关的 5-10 个片段，节省 90% Token，突破上下文限制

### 2. 强制暂停机制 (防止失控)
- **触发**: 每 50 章自动锁死
- **输出**: 中期报告（人物/伏笔/战力评估）
- **解锁**: 用户确认"继续"或"调整"后才解锁

### 3. 自动记忆系统
- **人物卡**: 新增角色自动添加，状态变化自动更新
- **伏笔表**: 埋设/回收自动标记，定期提醒未回收伏笔
- **时间线**: 记录事件顺序，检测矛盾
- **文风特征**: 提取前 10 章风格，后续章节模仿

### 4. 防崩坏机制
- **OOC 检测**: 生成后对比人物卡，评分低自动重写
- **战力监控**: 记录战斗对比，检测不合理越级
- **逻辑检查**: 检查时间线/因果关系

### 5. 批量操作
- 批量大纲 / 批量润色 / 批量续写 / 批量导出

## 🛠️ 快速开始

### 前置条件
- 已安装 **Hermes Agent**
- 已配置 **讯飞星辰** API (模型: `astron-code-latest`)

### 安装步骤
1. **克隆本仓库**:
   ```bash
   git clone https://github.com/2306192492-coder/novel-writer-skill.git
   ```

2. **复制到 Hermes 技能目录**:
   ```bash
   cp -r novel-writer-skill ~/.hermes/skills/creative/novel-writer
   ```

3. **刷新 Hermes**:
   ```bash
   hermes update
   ```

### 使用示例

#### 开书 (从零开始)
```text
你: "写一本主角能看到气运的修仙小说"

技能自动执行:
1. 解析需求 → 题材/风格/核心梗
2. 生成大纲 → 3 幕式 + 分卷规划
3. 人物设定 → 主角/反派/配角
4. 世界观 → 力量体系/地理/势力
5. 创建项目目录 → ~/Novels/书名/
6. 初始化记忆库 → 人物卡/伏笔表/时间线
```

#### 章节生成
```text
你: "写第一章" 或 "继续"

技能自动执行:
1. 检查强制暂停 → 如刚到 50 章节点且未确认，锁死
2. 向量检索 → 搜索相关设定/伏笔
3. 加载上下文 → 前 3 章 + 检索结果 + 人物卡
4. 生成正文 → 2100-2500 字
5. 三重自检 → OOC/逻辑/战力
6. 更新记忆库 + 向量索引
7. 自动保存
```

#### 设定整理
```text
你: "整理前 50 章设定"

技能自动执行:
1. 全量扫描 → 读取前 50 章
2. 更新人物卡/伏笔表/时间线
3. 生成中期报告
4. 建议后续方向
```

## ⚙️ 配置说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| **字数/章** | 2100-2500 字 | 可调整 |
| **暂停间隔** | 50 章 | `/set pause_interval N` |
| **检索片段数** | 5 个 | `/set search_topk N` |
| **保存路径** | `~/Novels/` | 自动创建 |

## 📝 常用指令

| 指令 | 功能 |
|------|------|
| "写一本 [题材] 小说，[核心梗]" | 开书 |
| "写第一章" / "继续" | 生成章节 |
